module Mat: ArrayMatrix
type t =
| Matrix of float array array
| Vector of float array

The type of an array matrix.

val rep_ok : t -> t

AF: |[|a11;a12;...;a1n|]; [|a21;a22;...;a2n|]; ...; [|am1;am2;...;amn|]| represents the m by n matrix with aij as entries. RI: The length of each array in the array of float arrays should be the same length. 0x0 matrices allowed.

val same_dim : t -> t -> bool

same_dim a b checks if a and b have the same dimensions. Requires: a and b are valid vectors or matrices.

val op : t -> t -> (float -> float -> float) -> t

op a b f is f applied to each pair of corresponding elements from a and b.
