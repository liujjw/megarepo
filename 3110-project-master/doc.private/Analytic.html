Module Analytic

module Analytic: sig .. end

Analytic functions: algebraic functions which can be approximated locally by power series.

type t

The type of an analytic function.

val make : Ast.f_expr -> t

make f is the t representation of an f_expr f.

val root : int -> t -> float

root e t is a root of t up to e digits. Requires: t is a single variable function that is supported. 1 < e <= 16 Raises: failure("Could not find a root.") if no root could be found.
