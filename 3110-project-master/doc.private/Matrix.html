Module Matrix

module Matrix: sig .. end

Matrix abstraction from matrix algebra: mxn grids of values.

type t

The type of a matrix.

val add : t -> t -> t

add m1 m2 is the sum of matrices m1 and m2. Requires: Matrices have correct dimensions for addition.

val subtract : t -> t -> t

subtract m1 m2 is the difference of matrices m1 and m2. Requires: Matrices have correct dimensions.

val mult : t -> t -> t

mult m1 m2 is the matrix product of m1 and m2. Requires: Correct dimensions.

val scale : float -> t -> t

scale r m1 is the scalar multiple of m1 by r.

val transpose : t -> t

transpose m1 is the transpose of m1.

val id : int -> t

id n is the nxn identity matrix.

val rand : int -> int -> int

rand i n m is the random nxm matrix with ints from 0 to i.
