Contains types for an abstract syntax tree.

type bop =
| Add
| Mult
| Sub
| Div
| Exp

The type of binary operators.

type uop =
| Sin
| Cos
| Tan
| ASin
| ACos
| ATan
| LogE
| Root

Type of unary operators .

type mop =
| Tran
| Rand

Type of unary matrix operators.

type num = {
   converted : bool;
   value : float;
}

The type of "Numbers".

type f_expr =
| FVarM of string * int
| FVarS of string
| FConst of num
| FBinop of bop * f_expr * f_expr
| FUop of uop * f_expr
| FNil

The type of the AST along with expr.

| Matrix of expr list

| Let of string * expr
| Var of string

| EvalAnalyticFuncM of string * expr list

| Matop of mop * expr list

to_int s is the int representation of a num s Requires: s.value is between 1 and 16.
